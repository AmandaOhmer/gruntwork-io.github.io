Support Plans for Startups to Enterprises

Choose the support program that’s right for your team:

Community Support

Get support from other Gruntwork customers in a shared chat room. Gruntwork engineers will also monitor the community channel, but without any guaranteed response time.

  • Live Chat with Fellow Gruntwork Community Members
  • Custom module development available

Professional Support

Chat directly with Gruntwork engineers in real-time via a private shared Slack channel or email, with guaranteed response times.

Everything in Community Support, plus:

  • Chat real-time with Gruntwork engineers in Slack
  • Send support requests via email
  • Guaranteed Timely Responses (Service Level Agreement)

Enterprise Support

Chat directly with Gruntwork engineers in real-time via a private shared Slack channel, email, or phone/video calls, with same-day guaranteed response times. Leverage the Gruntwork team for code reviews, architecture reviews, on-site training, and prioritized bug-fixes.

Everything in Professional Support, plus:

  • Work Through Issues via Live Phone/Video
  • Code Reviews
  • Architecture Reviews
  • Prioritized Bug fixes
  • DevOps Bootcamp available
